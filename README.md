# movie
此项目仅为个人兴趣爱好，开发电影网站爬虫爬取网站内容, 利用Lucene建立索引，SmartChineseAnalyzer做为中文分词器，
然后开发web端程序，提供检索页面，同时为用户提供推荐

Crawler是爬虫程序，爬取电影网站内容，建立索引
movieWeb是web端程序（springMVC开发），提供检索服务
